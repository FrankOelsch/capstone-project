- React Hooks
- React Router
- Styled Components
- React Testing Library
- Jest
- Canvas

### `Project Setup`

1. Clone this repository
2. Install all npm dependencies
   `npm install`
3. To run the app in development mode
   `npm start`
   then open http://localhost:3000 to view it in the browser
4. To run unit and component tests
   `npm test`
